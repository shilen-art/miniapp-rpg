# miniapp-rpg



Idle изометричная RPG-игра как Telegram Mini App.



## Стек



- Vite

- React 18

- TypeScript

- pixi.js v7

- @pixi/react v7

- Telegram WebApp API (через `window.Telegram.WebApp`)



## Локальный запуск



```bash

npm install

npm run dev

```



Приложение откроется по адресу http://localhost:5173.



## Git — базовые команды



Инициализация репозитория (выполнить один раз в корне проекта):

```bash

git init

git add .

git commit -m "chore: initial project setup"

```



Добавление и коммит изменений:

```bash

git add .

git commit -m "feat: короткое описание изменения"

```



Подключение удалённого репозитория (пример для GitHub):

```bash

git remote add origin git@github.com:USERNAME/miniapp-rpg.git

git branch -M main

git push -u origin main

```
